# VS-Code版RtcBuilder Java用拡張機能
Visual Studio Code(VS-Code)で動作するOpenRTM-aist向けコンポーネント開発ツールの拡張機能です．
Java版RTコンポーネントの雛形コードを生成するための拡張機能です。
本拡張機能を使用するためには，VS-Code版RtcBuilder本体が必要です．

### 開発環境構築方法
ソースコードを配置したディレクトリで以下を実行してください．  
```npm ci```

### 使用ライブラリ
- Nunjucks 3.2.4
  - <https://mozilla.github.io/nunjucks/>
  - JavaScript用テンプレートエンジン
